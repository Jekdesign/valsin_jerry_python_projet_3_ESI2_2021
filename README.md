# valsin_jerry_python_projet_3_ESI2_2021


### Install
```
  [~]:$ cd repo_name
```

```
  [repo_name]:$ ./bin/activate or source ./bin/activate 
```

```
  [(env_virtuel) > repo_name]:$ pip install -r requirements.txt 
```

### Run
```
  [(env_virtuel) > repo_name]:$ make start
```

### Stop
```
  [(env_virtuel) > repo_name]:$ deactivate
```
